3. Basic Feature of the App

The fintech web app aims to help the user achieve a personalised savings goal in a given time period and also suggests low risk investment plans to boost the savings process.

It takes the inputs (average income, average expense, savings goal and goal date) from the user and suggests a savings plan or pattern which will help the user spend wisely.

And if the savings are too low, it gives various investment portfolios to invest in and the user can choose the level of risk he/she wants to take.
